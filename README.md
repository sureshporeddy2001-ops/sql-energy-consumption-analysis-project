# sql-energy-consumption-analysis-project
A SQL data analysis project examining global energy consumption patterns by analyzing country-level data on emissions, production, population, and GDP to generate meaningful insights and support data-driven decision-making
# Energy Consumption Analysis (SQL Project)

## Overview
This project analyzes global energy consumption data across countries.

## Data Covered
- Energy production
- Energy consumption
- Emissions
- Population
- GDP

## Objective
To find relationships between energy usage, economic growth, and environmental impact.

## Tools Used
- SQL

## Outcome
This project helps understand how energy consumption varies across countries and its impact on emissions and GDP.
